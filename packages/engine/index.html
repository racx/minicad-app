Commands & shortcuts

Draw

L / LINELine (chained; Enter to end)
PL / PLINEPolyline — type C to close
RECRectangle by two corners
C / CIRCLECircle — center, then radius (click or type)
A / ARCArc through three points: start, a point on it, end
T / TEXTText — point, height, string
DIM / DLIDimension — two points, then place the line (value updates live)
DIMTXTDimension text size — number, or A = automatic; applies to selected dims + future ones

Modify

M / MOVEMove selection: base point → destination
CO / COPYCopy (multiple; Enter to end)
RO / ROTATERotate: base point, angle (type ° or click)
SC / SCALEScale: base point, factor
O / OFFSETParallel copy: distance → object → side (lines, circles, arcs, polylines)
TR / TRIMTrim to cutting edges (empty Enter = all objects are edges)
EX / EXTENDExtend to boundary edges (empty Enter = all objects)
F / FILLETFillet two lines — radius 0 = sharp corner, >0 = rounded
MI / MIRRORMirror across a two-point line (asks to erase source)
S / STRETCHCrossing box → endpoints inside move, rest stays put
GripsDrag the blue squares on a selected object to reshape it
CH / CHLAYERMove the selection to another layer
Double-click textEdit its wording in place
E / ERASEErase selection (or Delete key)
Drag selectedMove objects directly (Esc while dragging = abort)
UUndo  ·  REDO to redo

Inspect / view

DI / DISTMeasure between two points
Z → EZoom extents (A = all)
WheelZoom at cursor
P / PANHand tool — drag to move the view, Enter/Esc to exit
Middle-dragPan (or hold Space + drag)
